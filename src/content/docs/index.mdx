# 项目概述
本项目旨在帮助中国大陆地区的**国际版** [Minecraft](/docs/glossary#minecraft) 玩家发现服务器，并实现快速联机。  

项目所用的大多数构件均源自于[开源项目](/docs/glossary#开源)，  
项目的开发成本较低，项目的核心代码最后也会做开源处理。  

本项目的理念较新，当前市场上几乎没有类似的竞品，完全可以依靠后期提供增值服务创收。  

同时，由于项目可以高度和传统 [idc 模式](/docs/glossary#idc) 集成。  
即便宣发失败，项目也可以凭借特色服务参与其他 idc 厂商的竞争。
