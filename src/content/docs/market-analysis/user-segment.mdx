# 目标用户分析
我们主要的目标用户群体：  

#### 1. 有 MC 联机需求的玩家和服主
##### 特征
- 往往是学生，或者工作不久的年轻人，有一定的经济基础，对 MC 有情怀，愿意为更好的游戏体验付费。

##### 需求
- 快捷、稳定、上手即用的联机服务，最好还能方便的个性化定制游戏体验。  
- 有小伙伴陪他玩

##### 痛点
- 若用内网穿透或其他类似以网络工具形式的联机软件联机：  

    上手难度高，对玩家而言太不方便。  

- 若自建服务器：  

    费时费脑费钱，而很多用户只是想玩一阵子，不愿付出这么大成本。

- 若租用服务器：  

    会担心不会配置的问题，且服主单独花钱往往消费动力低。

- 找不到联机的小伙伴：  

    即便愿意为热爱付费，大部分情况下往往还没人陪他玩。

#### 2. 想要接入联机平台的其他 idc 企业
##### 特征
- 有一定的机子储备量
- 有基础的技术和运维能力

##### 需求
- 极度需要宣发来增加销量
- 渴望更高的用户消费动力

##### 痛点
- 没有好的，精准定位用户群体的宣发渠道
- 用户黏性低，没人玩了服主就不开服了，所以大部分服务器只开第一个月。
- 大量算力资源闲置，却不知道该怎样充分利用
